Cobblestone

A lightweight Jekyll based starter framework

Hosted and maintained on Github

See the project

Sass inside

Buildt and ships with Sass, pre-set and customizable variables for main elements.

Responsive

Ready to use mobile first CSS with media queries. All devices and IE8+ compatible.

Lightweight

Cobblestone ships with minimum modules. You have the basics, you build the rest.

Features

Normalize

Normalize CSS by Nicolas Gallhager for improved cross-browsers rendering

Typography

Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

Variables

Les variables peuvent être changées pour personnaliser le framework rapidement. Utilisées telles quelles, elles contituent une bonne base pour commencer à coder rapidement.

Layout

Gouttière

Valeur : $gutter: 20px;

Définit la largeur de la gouttière entre chaque élément de la grille.

Breakpoints

$bpXX

Définissent les quatre breakpoints de la grille.
